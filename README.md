# XiaoHH Design UI 脚手架

## 安装依赖

```bash
# 直接安装
npm install

# 使用中国内地境内镜像源/阿里源进行安装
npm install --registry=https://registry.npmmirror.com

# 使用中国内地境内镜像源/华为源进行安装
npm install --registry=https://mirrors.huaweicloud.com/repository/npm/
```

## 运行项目

```bash
# Mock 模式启动
npm run dev:mock

# 开发模式启动，菜单没有进行国内化
npm run dev
```

## 格式化代码

```bash
# 格式化所有代码
npm run format
```
